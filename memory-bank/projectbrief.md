# Project Brief

## Core Requirements

This project, genesis_02, is established with a foundational focus on strict code
quality and documentation standards.

### Key Objectives

1. Maintain highest code quality through strict linting
2. Ensure consistent development practices
3. Support effective AI collaboration
4. Maintain comprehensive documentation

### Project Standards

- All code and documentation must pass strict linting rules
- Package versions must only be managed via CLI tools
- Documentation must be current and thorough
- AI collaboration must follow established memory bank practices
